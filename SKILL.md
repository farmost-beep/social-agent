# 社交关系AI管家

一个本地运行的AI管家，自动追踪所有社交关系中「最近干了什么」和「下一步该干什么」，帮你拟好消息，推到该去的地方。

## 前置条件

- Python 3.9+
- Claude Code（AI拟稿引擎）
- 微信推送：cc-connect（可选）

## 触发场景

用户提到"社交关系"、"关系管家"、"帮我记住"、"联系人"、"记一下"、"拟消息"、"社交"、"人情世故"、"好久没联系"、"该联系谁"等与人际关系管理相关的词句时触发。

## 核心规则

涉及联系人数据的查询，**必须读 `contacts.json` 真实数据**，禁止从对话上下文猜答案。

数据修改用 `src/cli.py` 命令，不要直接编辑 JSON。

## 交互流程

### 1. 记录互动
```
用户：记一下：和张总聊了项目进展
→ 自动查找/创建联系人 → 记录时间线 → 提取待办
```

### 2. 查询联系人状态
```
用户：张总最近咋样
→ 查询时间线 → 返回最近互动+待办
```

### 3. AI拟稿
```
用户：给张总拟条消息
→ 提取上下文 → 生成草稿 → 确认→发送
```

### 4. 检查需跟进的关系
```
用户：最近该联系谁
→ 分析所有关系 → 列出14天+未联系的冷却关系
```

### 5. 查看待办
```
用户：最近有啥要跟进的
→ 列出P0/P1优先待办
```

## 子命令

在项目根目录下执行：

| 命令 | 说明 |
|:----|:-----|
| `python3 src/cli.py status` | 查看所有联系人 |
| `python3 src/cli.py todos` | 查看待办 |
| `python3 src/cli.py dashboard` | 仪表盘 |
| `python3 src/cli.py adjust` | 强度调整建议 |
| `python3 src/cli.py birthdays` | 生日提醒 |
| `python3 src/agent.py --chat "消息"` | 自然语言交互 |

## 数据目录

所有数据存储在 `data/` 目录下（可配置），JSON格式，本地存储不上云：

```
data/
├── contacts.json      # 联系人库
├── timeline.json      # 时间线
└── todos.json         # 待办队列
```
